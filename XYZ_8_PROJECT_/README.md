# Futuristic AI Solutions Landing Page

Una landing page interactiva y futurista sobre soluciones de IA, diseñada con estilo Bento y una paleta de colores personalizada.

## Características

- **Estilo Bento**: Diseño organizado en tarjetas y grids para una experiencia visual atractiva.
- **Paleta de Colores**: #6D1615, #D0B9BA, #096266, #739899.
- **5 Secciones**: Hero, Características, Servicios, Galería, Contacto.
- **Interactividad**: Animaciones de scroll, partículas en canvas, efectos hover, y formulario funcional.
- **Responsive**: Adaptable a diferentes tamaños de pantalla.

## Tecnologías Utilizadas

- HTML5
- CSS3 (con animaciones y gradientes)
- JavaScript (ES6+)

## Estructura de Archivos

- `index.html`: Estructura principal de la página.
- `style.css`: Estilos CSS con diseño Bento y paleta de colores.
- `main.js`: JavaScript para interactividad y animaciones.

## Cómo Ejecutar

1. Abre `index.html` en un navegador web moderno.
2. Para un mejor rendimiento, sirve los archivos desde un servidor local.

## Funcionalidades Interactivas

- **Hero Canvas**: Partículas animadas conectadas.
- **Scroll Reveal**: Animaciones de entrada al hacer scroll.
- **Hover Effects**: Transformaciones en botones y tarjetas.
- **Galería**: Clic en items para simular navegación.
- **Formulario**: Envío simulado con validación.
